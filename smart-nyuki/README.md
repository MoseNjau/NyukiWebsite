# Smart Nyuki

This project is a React application built using Vite and styled with Tailwind CSS. It serves as a template for creating a modern website that showcases various components such as a header, hero section, features, about section, services, team, testimonials, contact form, and footer.

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd smart-nyuki
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000` to view the application.

## Project Structure

The project is organized as follows:

- `src/`: Contains the source code for the application.
  - `main.tsx`: Entry point of the React application.
  - `App.tsx`: Main application component.
  - `components/`: Contains all reusable components.
  - `assets/`: Contains asset files (images, icons).
  - `styles/`: Contains global styles and Tailwind CSS imports.
  - `types/`: Contains TypeScript interfaces and types.

- `public/`: Contains static assets served directly.

- `index.html`: Main HTML template for the application.

- `package.json`: Configuration file for npm.

- `tsconfig.json`: TypeScript configuration file.

- `vite.config.ts`: Vite configuration file.

- `tailwind.config.js`: Tailwind CSS configuration file.

- `postcss.config.js`: PostCSS configuration file.
